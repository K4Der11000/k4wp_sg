لتشغيل البرنامج، اتبع الخطوات التالية:

1. تثبيت المتطلبات

افتح سطر الأوامر (Command Prompt) أو Terminal، ثم قم بتثبيت المكتبات المطلوبة باستخدام:

pip install requests cloudscraper beautifulsoup4 pillow

2. تشغيل البرنامج

بعد تثبيت المكتبات، احفظ الكود في ملف باسم wordpress_scanner.py، ثم شغّله باستخدام:

python wordpress_scanner.py

3. إدخال رابط الموقع

عند تشغيل البرنامج، ستظهر لك الواجهة الرسومية (GUI)، أدخل رابط موقع WordPress الذي تريد فحصه، ثم اضغط على زر "فحص".

4. عرض النتائج وحفظها

سيتم عرض نتائج الفحص مباشرة في النافذة.

يمكنك حفظ النتائج في ملف HTML عبر الضغط على زر "حفظ النتائج".
